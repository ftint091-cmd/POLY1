# Polymarket Copy Trader Bot 🤖

Бот для автоматического копирования ордеров с Polymarket с веб-интерфейсом управления.

---

## Описание

**Polymarket Copy Trader** — это Python-приложение на базе FastAPI, которое:

- 🔍 **Отслеживает** ордера заданного кошелька Polymarket через Polymarket CLOB API
- 📋 **Копирует** каждый новый ордер на ваш аккаунт с настраиваемым множителем размера
- 🖥️ **Предоставляет** веб-интерфейс для управления ботом и просмотра статистики
- 🛡️ **Идемпотентен** — каждый ордер копируется не более одного раза

---

## Возможности

- Периодический опрос ордеров целевого кошелька (настраиваемый интервал)
- Автоматическое размещение зеркальных ордеров через `py-clob-client`
- Настраиваемый множитель размера ордера (`COPY_MULTIPLIER`)
- Веб-интерфейс с тёмной темой — запуск/остановка бота, просмотр истории, отмена ордеров
- Логирование всех действий через стандартный Python-логгер
- REST API для интеграции с другими системами
- Развёртывание через Docker Compose

---

## Требования

- Python 3.11+
- Аккаунт Polymarket с API-ключами (key, secret, passphrase)
- Приватный ключ кошелька (для подписи ордеров)
- Сеть Polygon (Chain ID 137)

## Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/ftint091-cmd/POLY1.git
cd POLY1
```

### 2. Создать файл `.env`

```bash
cp .env.example .env
```

Заполните `.env` своими данными (см. раздел «Конфигурация»).

### 3. Установить зависимости

```bash
cd backend
pip install -r requirements.txt
```

### 4. Запустить сервер

```bash
cd ..
python -m uvicorn backend.main:app --host 0.0.0.0 --port 8000 --reload
```

Откройте браузер: [http://localhost:8000](http://localhost:8000)

---

## Развёртывание через Docker

```bash
cp .env.example .env
# Отредактируйте .env

docker compose up --build -d
```

Приложение будет доступно на порту `8000`.

## Конфигурация

Все параметры задаются в файле `.env`:

| Переменная               | Описание                                     | Значение по умолчанию          |
|--------------------------|----------------------------------------------|--------------------------------|
| `POLYMARKET_API_KEY`     | API-ключ Polymarket                          | —                              |
| `POLYMARKET_API_SECRET`  | Секрет API                                   | —                              |
| `POLYMARKET_PASSPHRASE`  | Passphrase API                               | —                              |
| `PRIVATE_KEY`            | Приватный ключ кошелька                      | —                              |
| `TARGET_WALLET_ADDRESS`  | Адрес отслеживаемого кошелька                | —                              |
| `CLOB_API_URL`           | URL Polymarket CLOB API                      | `https://clob.polymarket.com`  |
| `CHAIN_ID`               | ID сети (Polygon = 137)                      | `137`                          |
| `COPY_MULTIPLIER`        | Множитель размера ордера (1.0 = 100%)        | `1.0`                          |
| `POLL_INTERVAL_SECONDS`  | Интервал опроса целевого аккаунта (секунды)  | `10`                           |
| `HOST`                   | Хост для запуска сервера                     | `0.0.0.0`                      |
| `PORT`                   | Порт для запуска сервера                     | `8000`                         |

## API Endpoints

| Метод  | Путь                  | Описание                                    |
|--------|-----------------------|---------------------------------------------|
| GET    | `/api/status`         | Текущий статус бота                         |
| POST   | `/api/start`          | Запустить бота                              |
| POST   | `/api/stop`           | Остановить бота                             |
| GET    | `/api/orders/target`  | Ордера целевого аккаунта                    |
| GET    | `/api/orders/copied`  | История скопированных ордеров               |
| GET    | `/api/orders/own`     | Открытые ордера бота                        |
| POST   | `/api/config`         | Обновить конфигурацию (адрес, множитель, интервал) |
| POST   | `/api/cancel-all`     | Отменить все открытые ордера бота           |
| GET    | `/api/markets`        | Список доступных рынков                     |

### Пример: обновление конфигурации

```bash
curl -X POST http://localhost:8000/api/config \
  -H "Content-Type: application/json" \
  -d '{"target_address": "0xABC...", "copy_multiplier": 0.5, "poll_interval": 15}'
```

## Описание веб-интерфейса

- **Control Panel** — ввод адреса целевого кошелька, множителя и интервала; кнопки запуска/остановки и отмены ордеров
- **Bot Status** — текущее состояние: запущен/остановлен, количество скопированных ордеров, время последнего опроса
- **Target Orders** — таблица текущих ордеров отслеживаемого аккаунта
- **Copied Orders History** — история всех скопированных ордеров с временными метками
- **My Open Orders** — открытые ордера вашего бота
